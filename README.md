# Algorithms and Data Structures

This repository stands as a resource for various implementations of algorithms and data structures in a number of programming languages.

## Usage

```bash
# Python Scripts
pipenv run lint
pipenv run test

# JavaScript Scripts
npm run lint
npm run test

## PHP Scripts
composer run lint
composer run test

## Go Scripts
./go/scripts/lint.sh
./go/scripts/test.sh
```
